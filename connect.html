這篇文就是給你一個從現在開始努力的機會 雖然不可能一下子就趕上 // 畢竟人家也是有努力過的 但至少在學習的時候不會那麼的無力，這邊就是學長我當時上大學的暑假在看的影片

Lec01 資訊科技概論與程式設計(一)


裡面是教 C 語言，而老師是教 C++ ，但基本上語法大致上相同, 我已可以先做準備
影片中他們的 IDE (編寫軟體)是用 Dev C++ ，蔣教授則比較偏好 Code::Blocks (載點)
要哪一個你可以自己斟酌，寫程式這一科最重要的，終究還是自己的努力，加油吧！
