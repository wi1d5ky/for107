路程

HTML -> CSS -> Javascript -> jquery -> Bootstrap -> PHP 我現在學東西基本上都是先在網路上看基礎 & 要怎麼實做到網站上
等到有一定的能力後才選擇書籍 (不過我html和css都不曾看書學習XD, 所以也不是說非看書不可)

推薦

前面有提到我都是看網路上的資源來學習的,所以這邊就來推薦幾個網站

  • codecademy

    http://www.codecademy.com/
    學 HTML 和 CSS , javascript 裡面也有教, 但只是教如何使用 Bootstrap 並不是真正的 js 基礎, 所以我還在尋找適合的學習來源中...

  • F2Eclass

    http://F2Eclass.com
    裡面有很多前端工程的資源, 有時間可以去看看 應該可以學到很多東西

  • Mozilla webmaker

    https://webmaker.org/zh-TW
    Mozilla 為了推廣網際網路的開放, 特別研發了這個工具,讓學習的我們可以藉由修改他人的html來入門
    產生成就感, 進而慢慢對html等語言有興趣. 這裡 可以讓你更有方向地使用webmakere

  • W3school

    http://www.w3schools.com/
    很多人推薦這個網站來學習寫網頁, 但我自己是比較把它當辭典來用XD
    是個只有在需要查詢某些東西的用法時才會開的網站

一些資源

下面是我寫網頁時會用到的資源, 可以參考一下~ 去網路上找找你用得最順手的吧!

  • Flickr

    https://www.flickr.com
    常常寫網頁會需要用到許多圖片, flickr 就是一個很棒的存放的地方

  • 色碼表

    http://rhinejo.myweb.hinet.net/home/color/color-cord.html
    沒什麼特別的, 就是熱碼表而已XD 你可以自己去找用得順手的來使用~

  • 網頁字體

    http://www.google.com/fonts
    這是一個google提供的資源,裡面有很多網頁字體,有了這個就可以在自己的網站上用特別的字體
    不用再擔心使用者端有沒有裝某個字體了~ 這裡有使用教學, 可以參考一下

  • 網頁背景

    https://unsplash.com/
    每十天會更新一次的高畫質照片庫,很適合拿來當背景使用~

  • ICON下載

    http://free.com.tw/flaticon/
    就是一堆icon囉~

  • 設計師必備資源總匯

    http://resources.ogdesign.tw/
    一個培養美感的網站~

個人心得

目前我只有學到 Bootstrap 而已, 還沒辦法給太多的意見 不過上面那些語言, 學習過程中最重要的就是 "多練, 多實做" 常常使用就會寫得又快又好!
btw, 我不喜歡用 Frontpage 或 Dreamweaver 來寫網頁 我喜歡用 Notepad++ 來一個字一個字刻出來,這樣感覺比較爽(?)
最後一個小叮嚀: 請特別注意版權問題! 無論有甚麼樣的理由都不要觸法! NEVER!
